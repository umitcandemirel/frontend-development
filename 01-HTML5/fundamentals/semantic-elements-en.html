What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. HTML is used to display text, images, video, and other media types in web browsers.

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used to control the look and layout of web pages. CSS is used to define colors, fonts, sizes, and positions of HTML elements.
