# CSS Mini Projects

## English
A collection of small projects demonstrating core CSS concepts such as:
- Selectors
- Colors & Fonts
- Backgrounds
- Box Model
- Text Styling
- Float
- Position
- Margin
- Padding

Each project is designed to reinforce fundamental CSS knowledge with clean code practices.  
All projects are versioned and documented on GitHub for transparency and continuous improvement.
